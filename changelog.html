Changelog

All notable changes to the Arcyn Studios platform.

Milestone 3

Latest

Production Launch Preparation — July 2026

  • Added SEO improvements: dynamic meta, OG/Twitter tags, JSON-LD schemas, sitemap, robots.txt
  • Consolidated CSS to single render-blocking-free stylesheet
  • Added PWA support: service worker, install prompt, enhanced manifest
  • Added 404, offline, loading, and policy pages
  • Fixed mobile header overflow and touch target accessibility
  • Added keyboard navigation for search overlay
  • Improved color contrast and heading hierarchy

Milestone 2

Core Gaming Platform — June 2026

  • Launched category pages with dynamic banners and filtering
  • Added game detail pages with breadcrumbs, controls, and similar games
  • Implemented live search with keyboard navigation and recent queries
  • Added favorites system with localStorage persistence
  • Added toast notifications and loading skeletons
  • Added report modal and share functionality

Milestone 1

Platform Foundation — May 2026

  • Homepage with hero, featured games, trending, categories, and collections
  • Aurora Cyber design system with dark-first glassmorphism
  • Responsive layout with mobile drawer navigation
  • Dark/light theme toggle with system preference detection
  • Game library with 40+ titles across 10 categories
  • Shared game engine with canvas auto-sizing, input handling, and WebAudio SFX
